## LangGraph4j and Javelit integration

This module concerns integration with [Javelit] project. [Javelit] is an **open-source Java framework** designed for **data and backend engineers**. Its purpose is to help these engineers **deliver dynamic data apps with only a few lines of code**. Using [Javelit], users can **build (and deploy) powerful data apps very fast**.

> It could be considered the alter-ego of Phyton [Streamlit] framework in Java ecosystem

[Javelit]: https://javelit.io
[Streamlit]: https://streamlit.io
